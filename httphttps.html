HTTP stands for Hyper Text Transfer Protocol. It is a protocol for securing the communication between two systems, the browser and the web server. This is the protocol of the web. When your browser speaks to a web server, it speaks using HTTP. For example, the image of a duck below was requested by your browser using the HTTP protocol. The web server sent a response, again in HTTP, back to your browser which contained the duck image file. HTTP was initiated by Tim Berners-Lee at CERN in 1989 and has been widely accepted and used ever since.

HTTPS is HTTP with encryption. It stands for Hypertext Transfer Protocol Secure. For encryption it uses SSL (Secure Sockets Layer) or TLS (Transport Layer Security). If you took a look at the data packets in HTTP you could probably make out what they are since it will be in plain text. With HTTPS, this is not so since you will see what appears to be random characters. HTTPS works in collaboration with certificate authorities that evaluates the security certificate of the accessed website.
