DNS stands for Domain Name Server. When you type in google.com in to your browser, this information first goes to the DNS. The DNS takes the google.com input and directs your computer to the associated IP address. Google.com is much easier to remember then the IPv6 address 2001:4860:4860::8888. There are far too many domain names and associated IP addresses for one DNS server handle. So if the first DNS server you contact does not know the associated IP address, it will ask other DNS servers for this information. DNS severs are in a distributed hierarchal network divided into zones such as “.org”, “.com”, “.net” and so on.
